## Folder Structure

```
myclass/
├─ all/              # folder with student HTML files (e.g., Alice.html, Bob.html)
├─ public/           # optional: CSS/JS files
│   └─ style.css
├─ server.js         # Node.js script

```


Installation
```
npm init -y
npm install express
```

Run

```
node server.js
```
